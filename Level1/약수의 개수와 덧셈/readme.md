# 프로그래머스 코딩 테스트 연습 
## LEVEL 1.약수의 개수와 덧셈
https://programmers.co.kr/learn/courses/30/lessons/77884

#### 문제
- 두 정수 left와 right 가 매개변수로 주어집니다. left부터 right 까지의 모든 수들 중에서, 약수의 개수가 짝수인 수는 더하고, 약수의 개수가 홀수인 수는 뺀수를 return 하도록 solution함수를 완성해주세요.

#### 제한 조건
- 1 ≤ left ≤ right ≤ 1,000

#### 예시
##### 입력 및 출력
|left|right|result|
|----|----|----|
|13|17|43|
|24|27|52|
