# 프로그래머스 코딩 테스트 연습 
## LEVEL 1.3진법 뒤집기
https://programmers.co.kr/learn/courses/30/lessons/68935

#### 문제
- 자연수 n이 매개변수로 주어집니다. n을 3진법 상에서 앞뒤로 뒤집은 후, 이를 다시 10진법으로 표현한 수를 return 하도록 solutio 함수를 완성해주세요.

#### 제한 조건
- n은 1 이상 100,000,000 이하인 자연수입니다.

#### 예시
##### 입력 및 출력
|n|result|
|----|----|
|45|7|
|125|229|
