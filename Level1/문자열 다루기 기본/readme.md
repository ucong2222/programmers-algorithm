# 프로그래머스 코딩 테스트 연습 
## LEVEL 1.문자열 다루기 기본
https://programmers.co.kr/learn/courses/30/lessons/12918

#### 문제
- 문자열 s의 길이가 4 혹은 6이고, 숫자로만 구성돼있는지 확인해주는 함수, solution을 완성하세요.
- 예를들어 s가 "a234"이면 False를 리턴하고 "1234"라면 True를 리턴하면 됩니다.

#### 제한 조건
- s는 길이 1 이상, 길이 8 이하인 문자열입니다.

#### 예시
##### 입력 및 출력
|s|return|
|----|----|
|"a234"|false|
|"1234"|true|
